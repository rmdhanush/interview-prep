## RULES FOR YOURSELF

1. **DSA in Go only** — no Python, no Java. Build Go muscle memory.
2. **Time everything** — machine coding must finish in 90 min, HLD in 45 min.
3. **No peeking at solutions** for first 20 min of any DSA problem.
4. **Voice-record** yourself explaining system designs — listen back.
5. **Sunday = review day** — never start new topics on Sunday, only consolidate.
6. **Skip nothing** — if you can't solve a Hard, mark it and revisit in Week 12.
7. **One mock per week** from Week 8 onwards (Pramp, friend, or self-timed).
8. **Track streaks** — never miss 2 days in a row. One day off is OK, two is a habit.
